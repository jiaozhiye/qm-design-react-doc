## API

### UploadFile

| 参数      | 说明                         | 类型     | 默认值 |
| --------- | ---------------------------- | -------- | ------ |
| fileTypes | 限制上传附件的类型           | string[] | -      |
| fileSize  | 限制上传文件的大小，单位是 M | number   | -      |

说明：其他参数与 antd Upload 组件一样，参考地址：https://ant.design/components/upload-cn/#API
