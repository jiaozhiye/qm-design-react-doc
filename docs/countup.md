## API

### Countup

| 参数      | 说明                   | 类型                    | 默认值 |
| --------- | ---------------------- | ----------------------- | ------ |
| end       | 结束值，必要参数       | number                  | -      |
| delay     | 动画延迟时长，单位毫秒 | number                  | 0      |
| duration  | 动画持续时长，单位秒   | number                  | 2      |
| size      | 尺寸                   | medium \| small \| mini | -      |
| className | 自定义类名             | string                  | -      |
| style     | 自定义 css 样式        | CSSProperties           | -      |
| onEnd     | 动画结束时触发         | fucntion                | -      |
| onStart   | 动画开始时触发         | fucntion                | -      |
