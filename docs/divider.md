## API

### Divider

| 参数             | 说明                 | 类型                        | 默认值 |
| ---------------- | -------------------- | --------------------------- | ------ |
| collapse         | 展开/收起 状态       | boolean                     | -      |
| label            | 标题名称             | string                      | -      |
| size             | 尺寸                 | medium \| small \| mini     | -      |
| extra            | 摘要栏内容           | string \| JSXNode           | -      |
| className        | 自定义类名           | string                      | -      |
| style            | 自定义 css 样式      | CSSProperties               | -      |
| onCollapseChange | 展开/收起 切换时触发 | function(collapse: boolean) | -      |
